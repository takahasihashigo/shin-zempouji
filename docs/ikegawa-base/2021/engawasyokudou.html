2021-えんがわ食堂

01 : REPORT

活動報告

令和３年１２月１９日（日）に、仁淀川町善法寺にて、寺カフェとえんがわ食堂が開催されました。

「えんがわ食堂」は、いわゆる「こども食堂」の延長ですが、大人の方のカレー代金は次回のえんがわ食堂運営費に充てられます。ゆくゆくはこども食堂の定期開催を視野に入れています。

「寺カフェ」ではお茶を飲んでくつろいで頂きながら、子どもには自由な時間を、大人の皆様にはレスパイトケアの時間を作りたいと思い企画しました。福祉ネイルは、ネイルケアとハンドケアを通じて幸せで楽しい時間を提供しようという試みです。キッズスペースやフリーマーケットでの世代間交流と、リフレッシュタイムの提供を図りました。

当日はみぞれ交じりの天気でしたが、用意した１００食分のカレーは全てなくなり、１度帰った子どもがもう一度遊びに来るなど、楽しい時間を共有できました。

仁淀川町社会福祉協議会、高知県立大学社会福祉学部、善法寺お檀家様など、仁淀川町内外から総勢１８名のボランティア当日スタッフに参加していただきました。

池川ベースの活動は各種SNSを通じて発信しており、その様子を見た方々からご寄付も賜っております。神奈川県株式会社ダッドウェイ様、愛知県名古屋市本住寺様、四万十町窪川松鶴堂様、仁淀川町仁淀川アウトドアセンター様にはこの場を借りて厚く御礼申し上げます。

また、１２月２４日の高知新聞朝刊に当日の様子も掲載され、多くのお声を頂いております。

下記にボランティアスタッフ今橋様撮影の写真を掲載しながらの報告とさせて頂きます。
